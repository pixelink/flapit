# FlapIt Counter

This class let you send your individual data to the [flapit.com](https://www.flapit.com) API.

 - File based
 - Easy to configure
 - Send as many Data as you wish

## Usage
Include the class in your script and create a new instance of the class flapItCounter()
The class nees 3 mandatory parameter:
- GroupId
- Token
- Leading Flap shortcode [Flap It API documentation](https://www.flapit.com/de/api.html)

The fourth parameter is optional, it defines the file where the counter data is stored. By default it makes usage of a file named "counter.txt"
If you need to run different counters, just make sure to define alternative filenames.

#### Group id and token
You'll find that information in your [flapit.com](https://www.flapit.com) account
